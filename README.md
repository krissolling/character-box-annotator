# Character Box Annotator (React)

A modern, standalone React application for annotating character bounding boxes on images.

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:5173/

## Features

✅ Image upload with drag-and-drop
✅ Bounding box drawing
✅ Character picker with progress
✅ Image filters (brightness, contrast, etc.)
✅ Export to JSON
✅ Keyboard shortcuts (← → arrows)

## Usage

1. Upload an image
2. Enter text to annotate
3. Click "Start Annotating"
4. Draw boxes around characters
5. Export your annotations

See full documentation in the project folder.
